================================================================================
프로그램 개요
================================================================================
Kunhwa PDF Editor는 PDF 파일을 편집하고 텍스트를 추출할 수 있는 프로그램입니다.
허용된 사용자만 실행할 수 있으며, 맥어드레스 기반 인증 시스템을 사용합니다.

================================================================================
주요 기능
================================================================================
1. PDF 텍스트 추출
   - PDF 파일에서 텍스트를 추출하여 표시
   - 복사 및 편집 가능

2. 드래그 앤 드롭 지원
   - PDF 파일을 프로그램 창에 직접 드래그하여 열기
   - 파일 탐색기를 통한 파일 선택

3. 사용자 인증
   - 맥어드레스 기반 보안 시스템
   - 허용된 사용자만 프로그램 실행 가능

4. 썸네일 레이아웃 최적화 (v2.4 신기능)
   - 다양한 페이지 크기(A1~A4) 자동 인식
   - 썸네일 잘림 현상 방지
   - 페이지 비율에 따른 자동 크기 조정

================================================================================
사용 방법
================================================================================
1. 프로그램 실행
   - pdf_editor_v2.4_썸네일최적화버전.py 파일을 더블클릭하여 실행

2. PDF 파일 열기
   - "PDF 열기" 버튼 클릭하여 파일 선택
   - 또는 PDF 파일을 프로그램 창에 직접 드래그 앤 드롭

3. 텍스트 추출 및 편집
   - PDF 내용이 텍스트 영역에 표시됨
   - 텍스트를 선택하여 복사 가능
   - 텍스트 편집 후 저장 가능

4. 썸네일 관리
   - 좌측 패널에서 페이지 썸네일 확인
   - 썸네일 클릭으로 페이지 이동
   - 드래그 앤 드롭으로 페이지 순서 변경
   - Ctrl + 마우스 휠로 썸네일 크기 조정

5. 선택 페이지 저장 (v2.4 신기능)
   - 원하는 페이지만 선택하여 별도 저장
   - PDF 형식 또는 JPG 형식 선택 가능
   - 고화질(300 DPI) JPG 저장 지원

================================================================================
시스템 요구사항
================================================================================
- Windows 10 이상
- 별도 프로그램 설치 불필요

================================================================================
v2.4 업데이트 내용
================================================================================
1. 썸네일 레이아웃 최적화
   - 다양한 페이지 크기(A1~A4) 자동 인식 및 처리
   - 썸네일 오른쪽 잘림 현상 완전 해결
   - 페이지 비율에 따른 자동 썸네일 크기 조정

2. 사용자 경험 개선
   - 썸네일 선택 및 이동 성능 향상
   - 스크롤 영역 자동 최적화
   - 안전한 여백 설정으로 UI 안정성 향상

3. 선택 페이지 저장 기능 추가
   - 원하는 페이지만 선택하여 별도 저장 가능
   - PDF 또는 JPG 형식 선택 저장
   - 고화질(300 DPI) JPG 저장 지원
   - 다중 페이지 일괄 저장 기능

================================================================================
문제 해결
================================================================================
1. "접근 거부" 오류 발생 시
   - 관리자에게 문의하여 맥어드레스 등록 요청

2. 프로그램이 실행되지 않을 때
   - Windows Defender나 백신 프로그램에서 차단 여부 확인
   - 관리자 권한으로 실행 시도

3. 썸네일이 잘려 보일 때
   - 프로그램을 v2.4로 업데이트
   - 창 크기를 조정하여 최적화
