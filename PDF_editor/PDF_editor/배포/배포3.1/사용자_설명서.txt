Kunhwa PDF Editor v3.1 사용자 설명서 (배포용)

1) 설치/실행
- 폴더 전체를 원하는 위치로 복사한 뒤 Kunhwa_PDF_Editor_v3.1.exe 실행
- 별도 Python 설치 불필요(내장)
- EXE 아이콘은 회사 로고(kunhwa_logo.ico)로 적용됨

2) 구성 파일
- users.json.enc: 암호화된 사용자/권한 정보 파일(포함됨)
- data/kunhwa_logo.png, .ico: UI/아이콘 리소스

3) 기본 사용 흐름
- PDF 열기(Ctrl+O)  썸네일에서 선택/정렬/회전  저장(Ctrl+S) 또는 선택 저장(PDF/JPG)
- 새 창: 파일  새 창(빈) (상단 새 창 버튼도 동일 동작)
- 창 간 이동: Ctrl+C(복사)  대상 창에서 Ctrl+V(붙여넣기)
   환경에 따라 DnD(드래그앤드롭)가 제한될 수 있으므로 Ctrl+C/V 권장

4) 주요 기능
- PDF 열기/병합/저장/선택 저장(PDF/JPG, JPG 300DPI)
- 썸네일 기반 페이지 순서 편집, 회전(좌/우 90)
- 다중 창(파일 메뉴의 새 창) 지원, 창 간 페이지 이동(Ctrl+C/V)
- 사용자 관리(암호화 파일 기반): 사용자 목록/권한 관리
- 성능 모드: 고성능/균형/고품질 (대용량 PDF는 고성능 권장)

5) 저장 기본 경로
- 파일 저장/선택 저장 시 현재 열린 PDF 파일의 폴더를 기본 경로로 제안

6) 단축키
- Ctrl+O: 열기 / Ctrl+S: 저장
- Ctrl+]: 우회전 / Ctrl+[: 좌회전
- Ctrl+N: 새 창(빈)
- Ctrl+C / Ctrl+V: 페이지 복사/붙여넣기(창 간 이동 포함)
- Delete: 선택 삭제 / Ctrl+A: 전체 선택 / ESC: 선택 해제

7) 사용자 관리 안내
- 기본은 로컬 users.json.enc 사용
- 여러 PC에서 추가/삭제 필요 시, 공용 네트워크 경로의 users.json.enc를 사용(권장)
   관리자 비밀번호 확인 후 변경 허용(보안 권장)

8) 문제 해결
- DnD가 동작하지 않으면 Ctrl+C/V 사용
- 백신/보안 정책으로 실행 차단 시 속성에서 차단 해제 후 재실행
- users.json.enc가 없으면 사용자 관리 기능이 제한될 수 있음
