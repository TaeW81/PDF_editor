================================================================================
프로그램 개요
================================================================================
Kunhwa PDF Editor v3.1은 PDF 파일을 편집하고 관리할 수 있는 고급 프로그램입니다.
멀티창 지원, 교차 창 페이지 이동, 맥어드레스 기반 보안 시스템을 제공합니다.
허용된 사용자만 실행할 수 있으며, 관리자 권한으로 사용자 관리가 가능합니다.

================================================================================
주요 기능
================================================================================
1. 멀티창 지원 (v3.1 신기능)
   - 여러 PDF 파일을 동시에 편집
   - Ctrl+N: 빈 새 창
   - Ctrl+Shift+N: 파일 선택 후 새 창
   - 창 간 독립적인 작업 환경

2. 교차 창 페이지 이동 (v3.1 신기능)
   - 한 창의 페이지를 다른 창으로 드래그 앤 드롭
   - 창 간 페이지 복사 및 이동
   - 실시간 페이지 동기화

3. 고급 PDF 편집 기능
   - 페이지 회전 (우로90°, 좌로90°)
   - 빈페이지 삽입
   - 페이지 순서 변경 (드래그 앤 드롭)
   - 선택 페이지 저장
   - PDF 병합 및 분할

4. 텍스트 추출 및 편집
   - PDF 파일에서 텍스트 추출
   - 복사 및 편집 가능
   - 고품질 텍스트 인식

5. 썸네일 관리 시스템
   - 다양한 페이지 크기(A1~A4) 자동 인식
   - 썸네일 크기 조정 (Ctrl+마우스 휠)
   - 그리드 레이아웃 자동 최적화
   - 스크롤 영역 자동 조정

6. 드래그 앤 드롭 지원
   - PDF/이미지 파일 직접 드롭
   - 창 간 페이지 이동
   - 파일 병합 및 삽입

7. 성능 최적화 (v3.1 신기능)
   - 🚀 고성능 모드: 빠른 처리 속도
   - ⚖️ 균형 모드: 속도와 품질의 균형
   - 🎨 고품질 모드: 최고 품질 렌더링
   - 메모리 사용량 자동 최적화

8. 사용자 관리 시스템 (관리자 전용)
   - 사용자 목록 보기
   - 사용자 추가/제거
   - JSON 파일 편집
   - 사용자 백업 및 복원

================================================================================
사용 방법
================================================================================
1. 프로그램 실행
   - Kunhwa_PDF_Editor_v3.1_배포용.exe 파일을 더블클릭하여 실행
   - 맥어드레스 기반 인증 확인

2. 멀티창 사용
   - Ctrl+N: 빈 새 창 열기
   - Ctrl+Shift+N: 파일 선택 후 새 창 열기
   - 각 창은 독립적으로 PDF 편집 가능

3. PDF 파일 열기
   - "PDF 열기" 버튼 클릭하여 파일 선택
   - 또는 PDF 파일을 프로그램 창에 직접 드래그 앤 드롭
   - 여러 PDF 파일 동시 열기 지원

4. 페이지 편집
   - 썸네일 클릭으로 페이지 선택
   - Ctrl+클릭: 다중 선택
   - Shift+클릭: 범위 선택
   - Delete: 선택된 페이지 삭제
   - 드래그 앤 드롭으로 페이지 순서 변경

5. 교차 창 페이지 이동
   - 한 창의 썸네일을 다른 창으로 드래그
   - 페이지가 자동으로 대상 창에 복사/이동
   - 실시간 창 간 동기화

6. 고급 편집 기능
   - "우로90°"/"좌로90°": 페이지 회전
   - "빈페이지": 새 빈 페이지 삽입
   - "병합": 다른 PDF 파일 병합
   - "다중 병합": 여러 파일 동시 병합

7. 성능 설정
   - 메뉴바 → 성능 설정에서 모드 선택
   - 고성능/균형/고품질 모드 중 선택
   - 실시간 성능 모니터링

8. 사용자 관리 (관리자만)
   - 메뉴바 → 사용자 관리
   - 사용자 추가/제거/편집
   - 백업 및 복원 기능

================================================================================
키보드 단축키
================================================================================
- Ctrl+N: 새 창 (빈)
- Ctrl+Shift+N: 파일로 새 창
- Ctrl+O: PDF 열기
- Ctrl+S: PDF 저장
- Ctrl+]: 우로 90° 회전
- Ctrl+[: 좌로 90° 회전
- Ctrl+A: 전체 페이지 선택
- Ctrl+C: 선택 페이지 복사
- Ctrl+V: 페이지 붙여넣기
- Delete: 선택 페이지 삭제
- ESC: 선택 해제
- Ctrl+휠: 썸네일 크기 조정

================================================================================
마우스 제스처
================================================================================
- 클릭: 페이지 선택
- 더블클릭: 페이지 이동 대화상자
- 드래그: 페이지 순서 변경
- Ctrl+휠: 썸네일 크기 조정
- 휠: 페이지 스크롤
- Shift+휠: 가로 스크롤

문제 해결
================================================================================
1. "접근 거부" 오류 발생 시
   - 관리자에게 문의하여 맥어드레스 등록 요청
   - users.json.enc 파일 확인

2. 프로그램이 실행되지 않을 때
   - Windows Defender나 백신 프로그램에서 차단 여부 확인
   - 관리자 권한으로 실행 시도
   - 필요한 파일들이 EXE와 같은 폴더에 있는지 확인

3. 멀티창이 열리지 않을 때
   - Ctrl+N 또는 Ctrl+Shift+N 단축키 사용
   - 메뉴바 → 파일 → 새 창 선택

4. 교차 창 이동이 안 될 때
   - 두 창이 모두 열려있는지 확인
   - 드래그 앤 드롭이 활성화되어 있는지 확인

5. 성능이 느릴 때
   - 메뉴바 → 성능 설정 → 🚀 고성능 모드 선택
   - 메모리 사용량 확인
   - 불필요한 창 닫기
